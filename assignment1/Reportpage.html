Media queries is a css used to create responsive layouts for different devices using breakpoints is acheived by using:

media (min-width: 320px) and (max-width: 576px) : smaller devices

media (min-width: 425px) and (max-width: 768px) : mobile devices

media (min-width: 768px) and (max-width: 992px) : Tablet devices

media (min-width: 1024px) and (max-width: 1440px) : Desktop devices

Proximity In the code, you can see examples of proximity in the navigation menu where navigation items (Home, About, Services) are placed close to each other within a <ul> element. They are also visually grouped together in the header and also how the boxes are uniformly placed and structured (icon,heading,content) in Roles and Responsibilities and so on..

Similarity In the code, you can see example of similarity in the Homepage select menus: Instead of using text fields we used select menus and grouped inside together to get filter preferences also the buttons and icons used are all have similar color styles and shapes. also in terms of layout and structure is applied consistently throughout the "services" section. Each service item follows the same layout: an image on one side and a title with a description on the other side.

Closure: In the code, Closure is achieved by encapsulating each service item within a container with a background color. This creates a sense of completeness and separates each service from the others.

Continuation: The content within each service item is presented in a linear format from left to right (image followed by text). This continuation aids users in reading and understanding the content flow.
